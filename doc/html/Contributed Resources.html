Here you'll find useful resources contributed by VeraCrypt users.

Third party binaries:

  • Linux Ubuntu PPA provided by user "Unit 193" (build done by Launchpad):
    • https://launchpad.net/~unit193/+archive/ubuntu/encryption
  • Linux Armv7 GUI/console 32-bit build on ChromeBook by user haggster:
    • veracrypt-1.0f-1-setup-arm.tar.bz2

Tutorials:

  • http://schneckchen.in/veracrypt-anleitung-zum-daten-verschluesseln/:
    • German tutorial on VeraCrypt by Andreas Heinz.
  • http://howto.wared.fr/raspberry-pi-arch-linux-arm-installation-veracrypt/:
    • French HowTo for building VeraCrypt on Raspberry Pi Arch Linux by Edouard WATTECAMPS.
  • clonezilla_using_veracrypt_ver_1.1.doc:
    • Tutorial on using VeraCrypt in CloneZilla for accessing encrypted backups. By pjc123.
  • https://bohdan-danishevsky.blogspot.fr/2016/11/raspberry-pi-raspbian-installing.html
    • Tutorial on installing and using official VeraCrypt binaries on Raspberry Pi (Raspbian) by Bohdan Danishevsky.

Miscellaneous:
