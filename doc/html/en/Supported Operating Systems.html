Supported Operating Systems

VeraCrypt currently supports the following operating systems:
  • Windows 11 (x64 and ARM64)
  • Windows 10 version 1809 (October 2018 Update) or later (x64 and ARM64)
  • Windows Server 2019 or later (x64)
  • macOS 12 Monterey or later
  • Linux x86, x86-64, ARM64 (see the Downloads page for currently packaged distributions)
  • FreeBSD x86-64 (starting from version 14)
  • OpenBSD x86-64 (starting from version 7.8)
  • Raspberry Pi OS (32-bit and 64-bit)
Note:
VeraCrypt 1.26.15 is the last version that supports 32-bit Windows, Windows 10 versions before version 1809, and Windows Server 2016.
VeraCrypt 1.25.9 is the last version that supports Windows XP, Windows Vista, Windows 7, Windows 8, and Windows 8.1.
VeraCrypt 1.25.9 is the last version that supports Mac OS X versions from 10.9 Mavericks to 11 Big Sur.
VeraCrypt 1.24-Update8 is the last version that supports Mac OS X 10.7 Lion and Mac OS X 10.8 Mountain Lion.

Note: The following operating systems (among others) are not supported: Windows RT, Windows 2003 IA-64, Windows 2008 IA-64, Windows XP IA-64, and the Embedded/Tablet versions of Windows.
