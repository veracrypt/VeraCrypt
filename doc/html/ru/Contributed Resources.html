Полезные ресурсы, предоставленные пользователями VeraCrypt.

Сторонние бинарные файлы:

  • Linux Ubuntu PPA, предоставлено пользователем "Unit 193" (автор сборки – Launchpad):
    • https://launchpad.net/~unit193/+archive/ubuntu/encryption
  • Linux Armv7 GUI/консольная 32-разрядная сборка на ChromeBook, выполненная пользователем haggster:
    • veracrypt-1.0f-1-setup-arm.tar.bz2

Руководства:

  • http://schneckchen.in/veracrypt-anleitung-zum-daten-verschluesseln/:
    • Немецкое руководство по VeraCrypt, автор Andreas Heinz.
  • http://howto.wared.fr/raspberry-pi-arch-linux-arm-installation-veracrypt/:
    • Французское руководство по сборке VeraCrypt на Raspberry Pi Arch Linux, автор Edouard WATTECAMPS.
  • clonezilla_using_veracrypt_ver_1.1.doc:
    • Руководство по использованию VeraCrypt в CloneZilla для доступа к зашифрованным резервным копиям, автор pjc123.
  • https://bohdan-danishevsky.blogspot.fr/2016/11/raspberry-pi-raspbian-installing.html
    • Руководство по установке и использованию официальных бинарных файлов VeraCrypt на Raspberry Pi (Raspbian), автор Bohdan Danishevsky.

Разное:

  • vcsteg2.py: скрипт на Python, пытающийся скрыть том VeraCrypt внутри видеофайла (стеганография).
