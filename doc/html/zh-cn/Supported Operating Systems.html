支持的操作系统

VeraCrypt目前支持以下操作系统：
  • Windows 11
  • Windows 10
  • Windows Server 2016
  • Mac OS X 14 Sonoma
  • Mac OS X 13 Ventura
  • Mac OS X 12 Monterey
  • Linux x86、x86 - 64、ARM64（从Debian 10、Ubuntu 20.04、CentOS 7、OpenSUSE 15.1开始）
  • FreeBSD x86 - 64（从12版本开始）
  • 树莓派操作系统（32位和64位）
注意：
VeraCrypt 1.25.9是支持Windows XP、Windows Vista、Windows 7、Windows 8和Windows 8.1的最后一个版本。
VeraCrypt 1.25.9是支持从Mac OS X 10.9 Mavericks到11 Big Sur版本的最后一个版本。
VeraCrypt 1.24 - Update8是支持Mac OS X 10.7 Lion和Mac OS X 10.8 Mountain Lion的最后一个版本。

注意：以下操作系统（以及其他一些操作系统）不受支持：Windows RT、Windows 2003 IA - 64、Windows 2008 IA - 64、Windows XP IA - 64以及Windows的嵌入式/平板电脑版本。



另请参阅支持系统加密的操作系统部分。
