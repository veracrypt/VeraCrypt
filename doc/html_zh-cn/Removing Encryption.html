如何移除加密

请注意，VeraCrypt只能对分区和驱动器进行原地解密（对于系统分区/驱动器，选择系统 > 永久解密系统分区/驱动器；对于非系统分区/驱动器，选择卷 -> 永久解密）。如果您需要从文件托管卷中移除加密（例如，您不再需要加密），请遵循以下步骤：

  1. 挂载VeraCrypt卷。
  2. 将VeraCrypt卷中的所有文件移动到VeraCrypt卷之外的任何位置（请注意，文件将在移动过程中自动解密）。
  3. 卸载VeraCrypt卷。
  4. 像删除其他文件一样删除它（容器）。

如果不希望对非系统分区/驱动器进行原地解密，在这种情况下也可以遵循上述步骤1 - 3。

在所有情况下，如果遵循了步骤1 - 3，则可以执行以下额外操作：

  • 如果卷是分区托管的（也适用于USB闪存驱动器）
    1. 右键单击桌面上或开始菜单中的“计算机”（或“我的电脑”）图标，然后选择管理。此时应会出现“计算机管理”窗口。
    2. 在计算机管理窗口中，从左侧列表中选择“磁盘管理”（在存储子树中）。
    3. 右键单击要解密的分区，然后选择“更改驱动器号和路径”。
    4. 此时应会出现“更改驱动器号和路径”窗口。如果窗口中未显示驱动器号，请单击添加；否则，单击取消。

      如果您单击了添加，则在出现的“添加驱动器号或路径”窗口中，选择要分配给该分区的驱动器号，然后单击确定。
    5. 在计算机管理窗口中，再次右键单击要解密的分区，然后选择格式化。此时应会出现格式化窗口。
    6. 在格式化窗口中，单击确定。分区格式化完成后，将不再需要使用VeraCrypt挂载该分区即可保存或加载文件。
  • 如果卷是设备托管的
  1. 右键单击桌面上或开始菜单中的“计算机”（或“我的电脑”）图标，然后选择管理。此时应会出现“计算机管理”窗口。
  2. 在计算机管理窗口中，从左侧列表中选择“磁盘管理”（在存储子树中）。
  3. 此时应会出现“初始化磁盘”窗口。使用该窗口初始化磁盘。
  4. 在“计算机管理”窗口中，右键单击代表加密设备存储空间的区域，然后选择“新建分区”或“新建简单卷”。
  5. 警告：在继续操作之前，请确保您选择了正确的设备，因为其上存储的所有文件都将丢失。此时应会出现“新建分区向导”或“新建简单卷向导”窗口；请按照其说明在设备上创建新分区。分区创建完成后，将不再需要使用VeraCrypt挂载该设备即可保存或加载文件。
