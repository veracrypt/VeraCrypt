文档 技术细节 符号说明

符号说明

 

C 密文块
DK() 使用加密/解密密钥 K 的解密算法
EK() 使用加密/解密密钥 K 的加密算法
H() 哈希函数
i n位块的块索引；n取决于上下文
K 加密密钥
^ 按位异或运算（XOR）
⊕ 模2n加法，其中n是最左边操作数和结果值的位大小（例如，如果左操作数是1位值，右操作数是2位值，则：1 ⊕ 0 = 1；1 ⊕ 1 = 0；1 ⊕ 2 = 1；1 ⊕ 3 = 0； 0 ⊕ 0 = 0；0 ⊕ 1 = 1；0 ⊕ 2 = 0；0 ⊕ 3 = 1）
⊗ 二元域GF(2)上两个多项式模x128 + x7 + x2 + x + 1的模乘运算（GF代表伽罗瓦域）
|| 连接
