VeraCrypt系统文件与应用程序数据

注意：%windir% 是Windows的主要安装路径（例如，C:\WINDOWS）

VeraCrypt驱动程序

%windir%\SYSTEM32\DRIVERS\veracrypt.sys

注意：当VeraCrypt以便携模式运行时，此文件不存在。

VeraCrypt设置、应用程序数据和其他系统文件

警告：请注意，VeraCrypt不会对本节列出的任何文件进行加密（除非它对系统分区/驱动器进行加密）。

以下文件保存在文件夹 %APPDATA%\VeraCrypt\ 中。在便携模式下，这些文件将保存到您运行 VeraCrypt.exe 文件的文件夹中（即 VeraCrypt.exe 所在的文件夹）：

  • "Configuration.xml"（主配置文件）。
  • "System Encryption.xml"（在对系统分区/驱动器进行原地加密/解密的初始过程中使用的临时配置文件）。
  • "Default Keyfiles.xml"
    • 注意：如果未使用相应的VeraCrypt功能，此文件可能不存在。
  • "Favorite Volumes.xml"
    • 注意：如果未使用相应的VeraCrypt功能，此文件可能不存在。
  • "History.xml"（尝试作为VeraCrypt卷挂载或用作VeraCrypt卷宿主的最后二十个文件/设备的列表；此功能可以禁用 - 有关更多信息，请参阅 永不保存历史记录 部分）
    • 注意：如果未使用相应的VeraCrypt功能，此文件可能不存在。
  • "In-Place Encryption"（在对非系统卷进行原地加密/解密的初始过程中使用的临时配置文件）。
  • "In-Place Encryption Wipe Algo"（在对非系统卷进行原地加密/解密的初始过程中使用的临时配置文件）。
  • "Post-Install Task - Tutorial"（在VeraCrypt安装或升级过程中使用的临时配置文件）。
  • "Post-Install Task - Release Notes"（在VeraCrypt安装或升级过程中使用的临时配置文件）。

以下文件保存在文件夹 %ALLUSERSPROFILE%\VeraCrypt\ 中：

  • "Original System Loader"（在将VeraCrypt引导加载程序写入第一个驱动器磁道之前，对其原始内容进行的备份）。
    • 注意：如果系统分区/驱动器未加密，此文件不存在。

以下文件保存在文件夹 %windir%\system32（32位系统）或 %windir%\SysWOW64（64位系统）中：

  • "VeraCrypt System Favorite Volumes.xml"
    • 注意：如果未使用相应的VeraCrypt功能，此文件可能不存在。
  • VeraCrypt.exe
    • 注意：仅当启用系统收藏卷的挂载时，此文件夹中才会有该文件的副本。
